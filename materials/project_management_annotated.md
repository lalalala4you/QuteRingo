## 会議で使うフレーズ (Meeting Phrases)

### 進捗報告
**原文:** 現在の進捗は予定通りです。来週までに第一段階が完了する見込みです。
**読み:** 現在  =  げんざい  (genzai)
の  (no)
進捗  =  しんちょく  (shinchoku)
は  (ha)
予定通り  =  よていどおり  (yoteidoori)
です  (desu)
。  (.)
来週  =  らいしゅう  (raishuu)
までに  (madeni)
第一段階  =  だいいちだんかい  (daiichidankai)
が  (ga)
完了  =  かんりょう  (kanryou)
する  (suru)
見込み  =  みこみ  (mikomi)
です  (desu)
。  (.)

**原文:** 申し訳ありませんが、少し遅れています。理由は仕様変更があったためです。
**読み:** 申し訳  =  もうしわけ  (moushiwake)
ありませんが  (arimasenga)
、  (,)
少し  =  すこし  (sukoshi)
遅れ  =  おくれ  (okure)
ています  (teimasu)
。  (.)
理由  =  りゆう  (riyuu)
は  (ha)
仕様変更  =  しようへんこう  (shiyouhenkou)
があったためです  (gaattatamedesu)
。  (.)


### 課題の共有
**原文:** 一つ気になる点があります。ここについてもう少し詳しく説明してもらえますか。
**読み:** 一つ  =  ひとつ  (hitotsu)
気に  =  きに  (kini)
なる  (naru)
点  =  てん  (ten)
があります  (gaarimasu)
。  (.)
ここについてもう  (kokonitsuitemou)
少し  =  すこし  (sukoshi)
詳しく  =  くわしく  (kuwashiku)
説明し  =  せつめいし  (setsumeishi)
てもらえますか  (temoraemasuka)
。  (.)

**原文:** この問題について、どのように対応すればいいでしょうか。
**読み:** この  (kono)
問題  =  もんだい  (mondai)
について  (nitsuite)
、  (,)
どのように  (donoyouni)
対応  =  たいおう  (taiou)
すればいいでしょうか  (surebaiideshouka)
。  (.)


### 提案
**原文:** こういう方法もあると思いますが、いかがでしょうか。
**読み:** こういう  (kouiu)
方法  =  ほうほう  (houhou)
もあると  (moaruto)
思い  =  おもい  (omoi)
ますが  (masuga)
、  (,)
いかがでしょうか  (ikagadeshouka)
。  (.)

**原文:** リスクを減らすために、事前にテストを行いませんか。
**読み:** リスク  (risuku)
を  (wo)
減ら  =  へら  (hera)
すために  (sutameni)
、  (,)
事前  =  じぜん  (jizen)
に  (ni)
テスト  (tesuto)
を  (wo)
行い  =  おこない  (okonai)
ませんか  (masenka)
。  (.)


### 調整
**原文:** 来週の火曜日はいかがですか。もし難しければ、別の日程を提案してください。
**読み:** 来週  =  らいしゅう  (raishuu)
の  (no)
火曜日  =  かようび  (kayoubi)
はいかがですか  (haikagadesuka)
。  (.)
もし  (moshi)
難しけ  =  むずかしけ  (muzukashike)
れば  (reba)
、  (,)
別の  =  べつの  (betsuno)
日程  =  にってい  (nittei)
を  (wo)
提案  =  ていあん  (teian)
してください  (shitekudasai)
。  (.)
